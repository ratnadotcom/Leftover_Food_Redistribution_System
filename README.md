#Leftover Food Redistribution System

## Project Description

This system helps reduce food waste by connecting donors with people in need.

## User Roles

* Donor
* Receiver
* Admin

## Features

* User Registration & Login
* Donor can add food
* Receiver can request food
* Admin can manage requests

## Technologies Used

* PHP
* MySQL
* HTML, CSS

## How to Run the Project

1. Install XAMPP
2. Start Apache & MySQL
3. Open phpMyAdmin
4. Create a database named: `leftover_food`
5. Import the SQL file from `/database` folder
6. Place project folder inside `htdocs`
7. Run:
   http://localhost/Leftover_Food_Redistribution_System
